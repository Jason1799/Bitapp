# Bitapp

简要说明：这是你的项目 `Bitapp`，包含一个示例 Python 应用程序。
11
## 依赖
- Python 3.8+
- 见 `requirements.txt`（示例：`streamlit`, `python-docx`）

## 快速开始
1. 在项目根目录打开终端：

```powershell
python -m venv venv
venv\Scripts\Activate.ps1    # PowerShell
pip install -r requirements.txt
```

2. 运行应用：

```powershell
python app.py
```

或如果是 Streamlit 应用：

```powershell
streamlit run app.py
```

## 项目结构
- `app.py`：主应用入口
- `requirements.txt`：依赖列表
- `output/`：程序输出目录

## 使用说明
- 编辑 `app.py` 中的逻辑，按需安装额外依赖。
- 运行后查看 `output/` 以获取生成的文件（如果有）。
